#### Очень простой Файловой Менеджер (Созданый с помощью ИИ) который умеет Копировать, Перемещать, Удалять файлы по маске с сохранением путей (подпапок) как в ПК версии Total Commander.
* Не умеет удалять/перемещять пустые папки
* Переключения между панелями через свайп.
* Стабильно работает с Android 2.3 до Android 8.0.
* Очень удобно если нужно переместить на другую папку все файлы с расширением .vtf из папки Materials от игр на движке Source...
